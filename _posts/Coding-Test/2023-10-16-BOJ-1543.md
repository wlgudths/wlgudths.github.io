# 🧐 아이디어

> 문제를 읽자마자 **replace** 함수가 딱 떠올랐다.  


# 📝 내 풀이
``` python
# 입력
words = input() # 문서
word = input() # 검색하고 싶은 단어

change = words.replace(word, '*') # 중복되지 않게 단어 > "*" 변환

# 출력
print(change.count('*'))
```

# 😳 풀이 리뷰

> **replace** 함수를 통해 단어를 숫자 혹은 특수문자로 바꿔 준 후 바꾼 문자를 **count** 해주면 되는 간단한 문제였다.
